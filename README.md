# rockscy/homebrew-tap

Homebrew formulas for projects under [github.com/rockscy](https://github.com/rockscy).

## Install

```sh
brew install rockscy/tap/spell
```

That's the same as:

```sh
brew tap rockscy/tap
brew install spell
```

## Formulas

| Formula | Description |
|---|---|
| [`spell`](Formula/spell.rb) | [AI command palette for your terminal](https://github.com/rockscy/spell) |
